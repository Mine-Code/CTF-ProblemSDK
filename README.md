# CTF-ProblemSDK

A Problem SDK for MineCodeCTF Server.

## Quick install

```bash
curl -sL https://raw.githubusercontent.com/Mine-Code/CTF-ProblemSDK/master/quick_install.sh | bash -s -- -
```

## Install

```plain
git clone https://github.com/Mine-Code/CTF-ProblemSDK
cd CTF-ProblemSDK
./install.sh
```
